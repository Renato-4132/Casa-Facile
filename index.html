Windows (Automatico)

La soluzione più rapida tramite installer pre-configurato.

1

Scarica il pacchetto installer:

Scarica Installer .zip
2

Estrai i file in una cartella locale (es. sul Desktop).

3

Esegui l'installer: configurerà database e icone automaticamente.

4

Nota: Eseguendo nuovamente l'installer, il sistema richiederà la creazione di un nuovo utente e configurerà una nuova utenza dedicata.

Installazione Manuale

Per Linux, Mac o utenti Windows che preferiscono il controllo diretto.

Fondamentale:

Scarica Python 3.13+. Durante l'installazione, devi assolutamente spuntare la casella "Add Python to PATH".

Il programma è auto-installante. Una volta configurato Python:

  1. Crea una cartella 'Nome Utenza' per il programma (Es. Famiglia_Rossi).
  2. Scarica e copia all'interno il file sorgente:
    Scarica Casa Facile.pyw
  3. Avvia il file: il software scaricherà da solo le librerie necessarie.

Per sicurezza, se l'auto-installazione fallisse, il comando manuale è:

pip install tkcalendar google-genai requests segno pypiwin32
